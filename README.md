# Simple Http Parser
## 这是一个简易的HTTP模拟请求桌面端APP，支持GET/POST携带请求头信息，可以通过填写请求标头，快速获取响应体，不受跨域限制，方便测试。
### 代码运行
- npm install
- npm run start
